    Social Media Automation Script

    Uses Python, Docker, OpenAI, Dropbox, and more

    This is a containerized application that uses Selenium and BeautifulSoup to perform web-scraping on popular Reddit posts, transforming them into text, then audio, and finally video.     Campus Open Street Mapping Tool

    Uses C++, Expat, Excel

    This project was a command-line application meant to simulate Google Maps functionality for getting around campus. It parses location information such as bike pathways, bus routes, and speed limits around my college town of Davis, CA. This data populates a class heriarchy which uses Disjkstra's Algorithm to find the best way to get around, either using a bike, the bus, or through walking. Times were matched against equivalent Google Map queries and were accurate within 15% depending on traffic. As part of a school project, I am unable to host the repository on Github due to plagiarism concerns from school administration.
